Three skill tracks

DF — Digital Forensics & IR

Forensics & Response

Evidence acquisition, file system triage, memory forensics, network analysis, timeline reconstruction — the full analyst workflow from first alert to written report.

  • Disk & file system triage with Sleuth Kit
  • Memory forensics — Volatility 3
  • Network PCAP analysis, JA3, DGA detection
  • Super-timeline with log2timeline/plaso
  • Full incident reconstruction (Phase 12)
RE — Reverse Engineering

Reverse Engineering

From raw hex to readable logic — x64 assembly fluency, binary format internals, Ghidra decompiler workflows, crackme progression, and custom protocol decoding.

  • x86/x64 assembly and calling conventions
  • PE/ELF format internals
  • Ghidra: P-code, decompiler, struct recovery
  • Three-level crackme progression
  • Custom C2 protocol reversal (Phase 10)
MA — Malware Analysis

Malware Analysis

Static triage, behavioral detonation, unpacking, YARA rule writing, capability mapping to ATT&CK — from "what is this file?" to a full behavioral report.

  • Static analysis — strings, FLOSS, capa, YARA
  • Dynamic analysis — strace, Frida, sandbox
  • Unpacking UPX, OEP recovery, IAT fix
  • Rootkits and persistence mechanisms
  • Ransomware kill-chain capstone
Everything is synthetic and defensive. Crackmes, simulated pcaps, fabricated binaries, and fictitious log files — built for learning DFIR/RE/MA skills. No live malware. Every artifact has documented educational purpose and maps to a specific curriculum phase.

Full Curriculum

~120 hours · all tracks

All 12 phases in sequence. Deep understanding of cross-phase connections. The recommended path for maximum skill transfer.

  1. Phases 1–3: foundations
  2. Phases 4–5: static analysis + Ghidra
  3. Phases 6–7: dynamic + unpacking
  4. Phases 8–9: memory + network
  5. Phases 10–12: protocol + capstone

Quick start

analyst@range:~ — getting started
# 1. clone the repository
git clone https://github.com/akrist-rai/blackvault
cd blackvault

# 2. open in browser (no server needed)
open index.html        # then click ▶ Enter Console

# 3. install tool prerequisites (Debian/Ubuntu)
sudo apt install sleuthkit gdb radare2 binutils upx-ucl \
     tcpdump wireshark-common strace ltrace hashcat john \
     binwalk foremost volatility3 zeek suricata yara

# 4. build the crackme artifacts
cd range/crackmes && make && cd ../..

# 5. verify artifact integrity
sha256sum range/disk/disk.img range/network/capture.pcap \
          range/memory/memory-sample.raw
